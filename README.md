## Develop

The following will:

* start the website
* launch a REPL
* start an nREPL server
* use rebel-readline

```shell
$ fileworthy.sh repl
```

Inside this REPL launch figwheel like so:

```clojure
user=> (start-figwheel)
```

In another terminal run the following to connect to the existing nREPL server:

```shell
$ lein repl :connect 
```
